# AI Excuse Generator

## Overview

AI Excuse Generator is a Python-based application that generates context-aware excuses for a variety of real-world situations using Google's FLAN-T5 language model. The project combines natural language generation with supporting utilities such as multilingual translation, text-to-speech conversion, PDF document generation, and synthetic chat creation to provide a more complete user experience.

## Features

* AI-generated excuses using the FLAN-T5-Large language model
* Text-to-speech conversion
* PDF supporting document generation
* Fake chat conversation generation
* Excuse history and favorites management
* Interactive user interface built with Google Colab widgets

## Technologies Used

* Python
* Google Colab
* Hugging Face Transformers
* FLAN-T5-Large
* PyTorch
* ipywidgets
* Pillow (PIL)
* Matplotlib
* gTTS
* FPDF

## Usage

1. Enter the scenario for which an excuse is required.
2. Select the desired output language.
3. Generate the excuse.
4. Optionally create supporting documents, audio output, or chat screenshots.
